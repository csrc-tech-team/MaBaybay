Workshop on Introductory Python Programming for Mathematics Teachers

The Bicol University invites you to attend the Workshop on Introductory Python Programming for Mathematics Teachers. Designed for high school and college math teachers, this workshop will empower educators with practical coding skills that can be directly applied in the classroom and enable them to integrate Python programming into their math curriculum. The workshop will be hosted by Bicol University in Legazpi City, Albay and will take place from February 13 to 15, 2025. Join us for this 3-day event that includes a hands-on experience where you will learn Python programming tailored for math educators!
