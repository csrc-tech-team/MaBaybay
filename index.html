CSRC Workshop 2025: Python Programming For Mathematics Teachers

Bicol University, Legazpi Albay - February 14-15, 2025

Welcome to the CSRC Workshop 2025 on Python Programming For Mathematics Teachers. This workshop, hosted at Bicol University, Legazpi Albay, will take place on February 14-15, 2025. Join us for a hands-on experience where you'll learn Python programming techniques tailored for math educators.
